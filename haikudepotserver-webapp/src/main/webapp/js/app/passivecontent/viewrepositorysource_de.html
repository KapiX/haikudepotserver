Die Repository-Quelle definiert eine URL, unter der Repository-Daten angefordert werden können. Für ein Repository existieren oft mehrere URLs, eine für jede Architektur.

Neben den eigentlichen Repository-Daten existiert eine Datei repo.info, die ein url Feld enthält. Es dient zur eindeutigen Identifizierung dieser Repository-Quelle und stellt die Verbindung zwischen den Daten des Haiku Depot Server und dem Repository in der HaikuDepot Desktop-Anwendung her.
