Der Repository-Source definiert eine URL, mit der die Datei der Repository unterladet könnten. Ofters wurde eine Mehrfach von URLs an einer Repository eingestellt und zwar eins pro Architektur.

Eine repo.info Datei steht neben dem Repository Datei zu verfügen und enthält ein url Feld. Das Feld identifiziert eine bestimmten Repository-Source. Das Feld verbindet auch die HaikuDepotServer Informationen mit der Repository in der HaikuDepot Desktop-Anwendung.
